```xdg-open```コマンドが似たような物とのこと。

```shell
# 入ってなかったので入れる
$ sudo apt install xdg-utils
$ xdg-open .
```

使えました。
エイリアス貼っておきます。

```shell
$ echo 'alias open='xdg-open'' >> ~/.bashrc
$ source ~/.bashrc
$ open .
```
